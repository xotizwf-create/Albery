# Еженедельный отчет по чату: порядок работы ИИ

Эта инструкция не заменяет активный промпт `chat_weekly_report`. Она фиксирует порядок источников и обязательную связь с настройками промтов.

## Главное правило

Перед формированием еженедельного отчета по одному чату:

1. Вызвать `get_ai_prompts` с `category_key = "chat_weekly_report"`.
2. Использовать активный `prompt_text` как строгий контракт анализа, JSON-структуры и человекочитаемого отчета.
3. Не формировать недельный отчет по чату только по этой инструкции.

## Данные для отчета

Для отчета по конкретному чату за `period_start` - `period_end` нужны:

1. Чат и `dialog_id`.
2. Ежедневные отчеты по этому чату только за те дни недели, где были сообщения.
3. Если ежедневного отчета за день с сообщениями нет, сначала сформировать его по инструкции `Формирование отчетов / Ежедневный отчет по чату`.
4. Недельный отчет предыдущей недели по этому же чату, если он есть.
5. События целей и открытые задачи, которые попали в ежедневные отчеты недели.

Дни без сообщений не требуют ежедневного отчета, не сохраняются как `no_data` и не блокируют недельный отчет. В `raw_input` недельного отчета можно указать список пустых дат как `skipped_empty_dates`, но без создания записей в `chat_daily_reports`.

## Обязательный результат

Активный ИИ-промт `chat_weekly_report` из раздела `Настройка промтов` определяет точную схему. В общем случае недельный отчет должен вернуть:

- `summary`;
- `dynamics_summary`;
- `next_week_action_plan`;
- `hanging_tasks_by_owner`;
- `not_done`;
- `hanging_goals`;
- `work_in_progress`;
- `risks/problems`;
- `recommendations`;
- остальные поля, которые требует активный prompt.

Главный блок - `next_week_action_plan`. Каждый пункт должен содержать:

- владельца;
- конкретное действие;
- объект/задачу;
- срок или `срок не указан`;
- критерий проверки;
- источник/дату.

## Сохранение

Итог сохранять через `save_chat_weekly_report`.

В `raw_input` фиксировать:

- `dialog_id`;
- `period_start`;
- `period_end`;
- какие ежедневные отчеты были использованы;
- использованный `prompt_id`/версию активного `chat_weekly_report`, если они доступны из `get_ai_prompts`.
