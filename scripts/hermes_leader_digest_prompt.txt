Порядок строго:

1) albery start_here_always_read_ai_instructions → albery get_context_guide. Действуй по живым инструкциям сервера.

2) albery list_leader_evaluations — без аргументов (период по умолчанию: прошлая пятница .. сегодня, т.е. пятница + понедельник + вторник + среда). Инструмент вернёт:
   - calls_count — сколько созвонов с оценкой руководителей за период;
   - by_leader — по каждому руководителю (Артур Степанян, Наталья Горюнова, Евгений Палей, Сергей Виноградов): массив items {call_date, topic, role (host/co_leader), verdict (good/minor_issue/issue), result_for_owner} и counts {good, minor_issue, issue}.

3) Правило тишины: если calls_count == 0 или by_leader пуст — верни ПУСТОЙ stdout. Telegram молчит, ничего не показываем.

4) Если данные есть — собери свод для собственника. По КАЖДОМУ руководителю, по которому есть оценки:
   - короткий вывод о том, как он вёл созвоны за период (держал ли планку и регламент, вёл по повестке или уходил в сторону, что сильно, где просел) — на основе result_for_owner из его items;

   - 1-2 самых показательных эпизода (со ссылкой на дату и тему созвона), без воды.
   Пиши живым человеческим языком, как аналитик собственнику, без канцелярита и без технических полей (verdict/role/JSON наружу не показывай — переводи в слова: «провёл хорошо», «небольшое замечание», «существенное несоответствие»).
