# Выгрузка транскриптов созвонов в Markdown

Когда владелец просит выгрузить транскрипт(ы) созвонов в md / markdown / «файлом»:

- **За ДЕНЬ или ПЕРИОД** («выгрузи созвоны/встречи/транскрипты за 28 мая», «за вчера», «за неделю») — всегда используй **export_zoom_transcripts_markdown** с `date_from` и `date_to`. Для одного дня ставь **одну и ту же дату** в `date_from` и `date_to` — тогда выгрузятся **ВСЕ встречи этого дня** (в дне часто 2–3 созвона), а не одна.
- **За ОДНУ конкретную встречу**, которую владелец явно указал, — используй **export_zoom_call_markdown** с `call_id`.
- **Никогда не выгружай только один созвон, когда просят «за день».** Если сомневаешься, сколько встреч в дне, сначала посмотри `list_zoom_calls` за эту дату, затем отдай их все одним файлом через `export_zoom_transcripts_markdown`.
- Оба инструмента сохраняют файл и возвращают `download_url`. Отдавай владельцу **ссылку** `download_url` (публичная, без логина), а не полный текст транскрипта — длинный текст обрезается клиентом.
