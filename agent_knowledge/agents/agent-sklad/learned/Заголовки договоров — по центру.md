Все заголовки договоров и юридических документов (название документа и заголовки разделов/пунктов) выравнивать ПО ЦЕНТРУ. В HTML для export_document оформляй заголовки тегами <h1>…</h1>…<h4>…</h4> — они автоматически центрируются; либо задавай явно style="text-align:center". Основной текст пунктов при этом остаётся выровненным по ширине (justify). НЕ печатай в тексте документа разметку [b]…[/b], ** ** или голые HTML-теги — для жирного используй <b>…</b>, для остального соответствующие HTML-теги; лишние теги в готовом Word недопустимы.
