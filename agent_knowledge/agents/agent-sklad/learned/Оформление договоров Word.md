При подготовке всех договоров и юридических документов Word через export_document использовать шрифт 12 pt для всего документа, включая заголовки; основной текст выравнивать по ширине с отступом первой строки 1,25 см; межстрочный интервал 1,5; тире использовать средние «–», не длинные «—»; реквизиты сторон оформлять двухколоночной таблицей без рамок; приложения начинать с новой страницы.
