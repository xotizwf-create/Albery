При подготовке юридических документов Word через export_document основной текст договора, соглашения, акта, приложения и иных официальных документов оформлять с выравниванием по ширине: для абзацев использовать style="text-align:justify; text-indent:1.25cm" там, где нужен отступ первой строки. Сохранять аккуратное ГОСТ-оформление: Times New Roman, межстрочный интервал 1,5, приложения с новой страницы, реквизиты сторон двухколоночной таблицей без рамок.
