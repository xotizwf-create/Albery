При подготовке ответов на отзывы, вопросы и чат WB для бренда Vontel работать по регламенту «Регламент по работе с отзывами» (Google Sheet 1MznLJDfNW-Kq1qPVeO1I7OBKz02acp1uNH50bNfNH8A). Обрабатывать в порядке: отзывы → вопросы → чат; рабочая периодичность — Пн–Пт. Отзывы: 4–5★ без текста (включая с фото) — отвечать дословно: «Здравствуйте! Спасибо, что делитесь впечатлениями! Для нас это бесценный источник вдохновения. Будем рады вашим новым заказам! С уважением, команда Vontel». 1–3★ — по каждому рекомендовать/фиксировать жалобу «Отзыв не относится к товару» независимо от содержания; затем готовить ответ по соответствующему ситуации шаблону. При прямом указании на брак и типе заказа «отказ» дополнительно указать действие «запросить возврат товара». 4–5★ с текстом — обязательно поблагодарить; при скрытом негативе не спорить, поблагодарить за обратную связь. Вопросы: маркетинговые (например просьбы о скидке, нерелевантные) — отклонить, ответа не готовить; технические/бытовые/пользовательские — короткий ответ по факту, без оценок конкурентов, завершать «С уважением, команда Vontel». Вопрос о размере упаковки — отвечать. Чат: нейтральный/без выраженного негатива — деловой терпеливый ответ по шаблону; спам или «тест» — не отвечать. При выраженном негативе (оскорбления, капслок, угрозы, крайние претензии, массовые повторы) не готовить самостоятельный ответ: запросить согласование руководителя, приложив скрин и пометку «СОГЛАСОВАНИЕ: негативный чат»; текст для клиента — только после согласования. При подготовке ответов всегда отмечать, когда по регламенту требуется жалоба, возврат, отклонение вопроса либо эскалация; не выдумывать характеристики товара — если их нет во входных данных, запросить уточнение.
