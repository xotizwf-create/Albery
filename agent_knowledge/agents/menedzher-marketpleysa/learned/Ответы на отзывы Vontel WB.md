При подготовке ответов на отзывы, вопросы и чат WB отвечать только по товарам бренда Vontel. Все отзывы и обращения по другим брендам пропускать: не готовить и не публиковать ответы. Для Vontel применять регламент «Регламент по работе с отзывами»: порядок отзывы → вопросы → чат; рабочие дни Пн–Пт. Отзывы 4–5★ без текста (включая с фото): отвечать дословно «Здравствуйте! Спасибо, что делитесь впечатлениями! Для нас это бесценный источник вдохновения. Будем рады вашим новым заказам! С уважением, команда Vontel». По 1–3★ фиксировать жалобу «Отзыв не относится к товару»; при браке и типе заказа «отказ» — запросить возврат. 4–5★ с текстом — поблагодарить, при скрытом негативе не спорить. Технические/бытовые вопросы — отвечать по факту и завершать «С уважением, команда Vontel»; маркетинговые — отклонять без ответа. При выраженном негативе в чате — эскалировать руководителю, самостоятельный ответ не готовить. Не выдумывать характеристики товара.
